# Password Generator

A modern Password Generator built with React, featuring customizable options for password length and inclusion of numbers and special characters. The app includes a stylish UI with smooth gradient backgrounds and custom round checkboxes for toggling settings. It also allows users to easily copy the generated password to the clipboard.

## Live Demo

[Generate Password](https://generate-password123.vercel.app/)

## Features

- **Customizable Length**: Set the desired length of the password.
- **Include Numbers**: Option to add numeric characters.
- **Include Special Characters**: Option to include special symbols.
- **Copy to Clipboard**: One-click copy functionality.
- **Modern UI**: Smooth gradient background and custom round checkboxes.
- **Responsive Design**: Works seamlessly on all devices.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Hosting**: Vercel
